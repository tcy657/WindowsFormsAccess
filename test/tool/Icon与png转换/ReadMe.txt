อ๊ณษศีฦฺ: 2005.8.12

ศํผฝ้ษ:
    AveIcon สวาปธ๖ะกวษตฤ ICO/PNG ปฅืชะกนคพ฿ฃฌำะมหหฮาพอฒปำริฺด๒ฟชลำด๓ตฤ Axialis IconWorkshop มหฃฌถ๘วาหฝ็รๆณฌผ๒ฝเฃฌึปะ่อฯถฏอผฦฌตฝดฐฟฺึะิูอฯถฏปุภดฃฌี๛ธ๖ืชปปนณฬพอหใอ๊ณษมหฃฌธม๎ศหะภฯฒตฤสวหปนึงณึ Alpha อจตภ.รปฯ๋ตฝีโรดะกตฤนคพ฿พนศปีโรดวฟด๓ฃฌฯ๑สวิฺื๖นใธๆฃฌอนนฃฌีๆสวผ๑ตฝด๓ฑใาหมหฃฌปนำะาปตใหสวรโทัตฤำด.
    ธ๖ศหพ๕ตรีโศํผตฤื๗ี฿ีๆBTฃฌำรDelphiพอตรมหฃฌพนศปด๓ฒฟทึืสิดถผฒปิฺฑ๊ืผืสิดภ๏รๆฃฌำฺสวทัพขตุด๒ฟชUEีาืสิด.ื๖อ๊บบปฏทขฯึืึฬๅฯิสพฒปศซฃฌผธดฮะธฤบ๓ทขฯึฮะงฃฌณฌณ๖ฑ฿ฟ๒ฒฟทึศซถผฯิสพฮชก๕ฃฌฮุฮุฃฌฦ๘หภลผมหฃฌำฺสวฟชสผิญทญา๋ฝ๘ะะห๕หฎฃฌณษฦทพอสวฯึิฺตฤีโธ๖ั๙ืำมห.
    าัึชฮสฬโ:อฯืงดฐฟฺึะตฤ"อฯืง"ถืึฟดฦ๐ภดฯ๑ยาย๋ฃฌำฆธรสวืึฬๅฮสฬโ.PS:ลผาัพญะธฤห๙ำะืึฬๅฮช Tahoma มหฃฌฟษปนสวีโธ๖ั๙ืำฃฌฦไสตลผาฒฒปฯ๋ฟดตฝีโั๙ตฤว้ฟ๖ฃฌฮุฮุ.
